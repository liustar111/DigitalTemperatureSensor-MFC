# DigitalTemperatureSensor-MFC

MFC-based desktop application for collecting and displaying digital temperature sensor data over a serial port.

## Contents

- `Digitaltemperaturesensor.sln`: Visual Studio solution
- `Digitaltemperaturesensor.vcxproj`: MFC project file
- `docs/figures/`: project figures and generation script

## Notes

- Build artifacts and local Visual Studio cache files are excluded from version control.
